## Setup

<Steps>
<Step title="Get your API Key">
1. Log in to your Dodo Payments dashboard
2. Navigate to Settings > API Keys
3. Copy your API key
</Step>

<Step title="Configure the Extension">
1. Open Raycast
2. Search for "Dodo Payments" and select any command
3. If not configured, you'll see an authentication screen
4. Enter your API key in the "API Key" field
5. Select your preferred API mode:
   - Test: For development and testing
   - Live: For production
6. Save the settings
</Step>
</Steps>
