title: Payments Integration with Code
description: This tutorial demonstrates integrating Dodo Payments into a Next.js app using the Node.js SDK for One-Time and Subscription Products. It utilizes the Dodo Next.js Minimal Boilerplate and covers essential steps to get started efficiently.
---


You can clone the Dodo NextJS Minimal Boilerplate we have used for the tutorial in the below link.

https://github.com/dodopayments/dodo-nextjs-minimal-boilerplate

We begin with the SDK initialization, ensuring you have the necessary setup to interact with Dodo's API. Then, we explain how to securely verify webhook signatures to handle payment events safely. You'll also learn how to fetch and display a list of products dynamically and complete a test payment to see the full workflow in action.
